常见问题

为您解答关于 OzonAssist 的一切疑问

使用本软件会不会导致封号？

软件完全模拟真实的人类操作流程，通过本地 API 与插件进行交互，不涉及篡改 Ozon 后台核心逻辑，安全性极高。我们已服务 1200+ 卖家，目前保持零风险记录。

是否支持 macOS 平台？

是的。OzonAssist 基于 Electron 开发，完美适配 macOS (包括 M1/M2/M3 及 Intel 芯片) 和 Windows 10/11 系统。

图片资料是存储在服务器上的吗？

不是。出于隐私考虑，所有的图片和 SKU 数据均存储在您本地的 SQLite 数据库中。除非您在使用过程中通过浏览器提交，否则任何数据都不会上传至云端。
